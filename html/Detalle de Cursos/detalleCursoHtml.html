Curso de HTML

Valor: $20.000

Tiempo de Dedicacion Necesario: 20hs

Curso práctico para aprender los fundamentos de HTML, el lenguaje base de la web. Incluye estructura de páginas, etiquetas principales, enlaces, imágenes, tablas y formularios, preparando al alumno para crear sitios web desde cero o avanzar hacia CSS y JavaScript.

Requisitos Previos: Ninguno

CONTENIDO POR CLASE

INTRODUCCION
  • Qué es HTML y su papel en la construcción de páginas web.
  • Estructura básica de un documento HTML.
  • Herramientas necesarias: editor de código y navegador.
  • Buenas prácticas de organización de archivos y carpetas.
UNIDAD 1
  • Etiquetas esenciales.
  • Encabezados, párrafos y texto enriquecido.
  • Enlaces y rutas: absolutas y relativas.
  • Listas ordenadas y desordenadas.
  • Imágenes y atributos principales .
UNIDAD 2
  • Tablas: creación y atributos.
  • Formularios: campos de texto, botones, selectores, validación básica.
  • Elementos semánticos de HTML5.
  • Buenas prácticas de accesibilidad.
