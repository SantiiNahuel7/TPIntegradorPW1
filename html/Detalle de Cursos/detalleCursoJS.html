Curso de Java Script

Tiempo de Dedicacion Necesario: 30hs

Este curso introduce a los estudiantes en el lenguaje JavaScript, fundamental para el desarrollo web moderno. Se aprenderán los conceptos básicos de programación, manipulación del DOM y creación de interactividad en páginas web, sentando las bases para avanzar hacia proyectos más complejos y dinámicos.

Requisitos Previos: HTML y CSS

$30.000

Contenidos Por Clase

INTRODUCCIÓN
  • ¿Qué es JavaScript y para qué se utiliza?
  • Relación de JavaScript con HTML y CSS.
  • Configuración básica del entorno de trabajo (navegador y editor de código).
UNIDAD 1
  • Variables y tipos de datos.
  • Operadores y expresiones.
  • Estructuras de control: condicionales y bucles.
  • Funciones y alcance de variables.
UNIDAD 2
  • Manipulación del DOM (selección y modificación de elementos).
  • Manejo de eventos (click, teclado, mouse).
  • Introducción a objetos y arreglos.
  • Creación de scripts dinámicos y prácticas con mini-proyectos.
